# CQUPT 校园网登录工具

重庆邮电大学校园网一键登录/注销工具，基于 ePortal 强制门户协议，支持设备伪装以绕过单设备限制。

## ⚠️ 重要提示

**如果你已经成功连上网络（例如通过网页端登录了校园网），请先通过网页端注销，再使用本程序登录。** 否则程序可能无法正确检测网络状态，导致登录失败。

## 功能特性

- **一键登录/注销** — 图形化界面，无需接触命令行
- **设备伪装** — 支持 PC / Mobile 两种模式，用 Mobile 模式可实现两台电脑共享一个账号上网
- **运营商选择** — 支持中国电信、中国移动、中国联通
- **凭据记忆** — 自动保存账号密码，下次打开自动填充
- **密码显示/隐藏** — 点击眼睛图标切换
- **开机自启** — 勾选后自动在 Windows 启动文件夹创建启动脚本
- **断线自动重连** — 默认每 5 分钟检测并自动重连，间隔可自定义
- **退出选项** — 关闭窗口时可选择先注销再退出，或直接退出保持在线

## 环境要求

- Windows 系统
- Python 3.12+（如果直接运行源码）
- 无需任何第三方依赖（仅使用标准库）

## 使用方法

### 从源码运行

```bash
# 普通启动（显示窗口）
python run.py

# 静默启动（最小化到任务栏，用于开机自启）
python run.py --silent
```

### 打包为 exe

```bash
pyinstaller --onefile --windowed --name "CQUPT校园网登录" --clean run.py
# 输出文件在 dist/CQUPT校园网登录.exe
```

## 使用步骤

1. 确保电脑已连接校园网 Wi-Fi 或有线网络
2. **确认当前处于未登录状态**（如已通过网页端登录，请先注销）
3. 打开程序，输入学号和校园网密码
4. 选择设备类型（PC / Mobile）和运营商
5. 点击「登录」按钮
6. 登录成功后窗口会自动最小化到任务栏，程序会在后台保持连接

## 设备伪装原理

校园网 ePortal 系统通常限制一个账号只能同时登录一台 PC 和一台手机。通过切换设备的 `callback` 参数和 `User-Agent`，本程序可以让同一台电脑伪装成手机端登录，从而实现两台电脑共享一个账号上网。

## 项目结构

```
config.py           # 常量定义（服务器地址、设备/运营商配置）
client.py           # CQUPT 登录客户端 — ePortal 两步登录/注销 HTTP 协议
config_manager.py   # 配置文件管理（%APPDATA%/dormnet_login/config.json）
autostart.py        # 开机自启管理（Startup 文件夹 .vbs 脚本）
gui.py              # tkinter 图形界面、多线程、断线重连
main.py             # 程序入口
run.py              # PyInstaller 打包入口
```

## 配置存储

配置文件保存在 `%APPDATA%/dormnet_login/config.json`，密码以明文存储（与参考实现 Kotlin DataStore 行为一致）。删除该文件即可重置所有设置。
