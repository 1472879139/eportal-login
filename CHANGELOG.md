# 更新日志

## v1.0.0 (2026-06-15)

CQUPT 校园网登录工具首次发布。

### 新增功能

- **可视化登录界面** — 基于 tkinter 的桌面窗口，无需接触命令行即可使用
- **一键登录/注销** — 连接校园网后点击登录完成认证，点击注销断开认证
- **设备伪装** — 支持 PC / Mobile 两种设备模式，用 Mobile 模式可实现双电脑共享账号上网
- **运营商选择** — 支持中国电信、中国移动、中国联通三种运营商
- **凭据记忆** — 登录时自动保存账号密码，下次打开自动填充
- **密码显示/隐藏** — 点击眼睛图标切换密码可见性
- **开机自启** — 勾选后在 Windows 启动文件夹创建静默启动脚本，开机自动运行
- **断线自动重连** — 默认每 5 分钟自动重连，间隔可自定义，保持持续在线
- **退出选项** — 关闭窗口时可选择先注销再退出，或直接退出保持在线
- **独立可执行文件** — PyInstaller 打包为单个 exe，无需安装 Python 环境

### 技术说明

- 语言：Python 3.12
- GUI：tkinter（标准库，零依赖）
- HTTP：urllib（标准库，零依赖）
- 打包：PyInstaller 6.x
